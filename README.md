Sales Analysis Dashboard - Power BI Project

🎯 Objective
To Analyze employee attrition rate and to identify key factors influencing employee attrition rate. 

📌 Project Overview
This project analyzes employee attrition data to identify key factors contributing to employee turnover. The Objective is to uncover patterns and provide data - driven insights that can help improve employee retention strategies.

🛠 Tools Used
Python
Pandas
Numpy
Matplotlib
Seaborn
Jupyter Notebook

📊 Key Steps Performed
Data Cleaning and preprocessing
Handling missing values
Exploratory Data Analysis (EDA)
Visualization of attrition trends
Correlation analysis between Variables

📈 Key Insights
Identified major factors influencing employee attrition
Observed relationship between salary level and attrition date
Department-wise variation in employee turnover
